⚙️ Backend & API
O núcleo da aplicação foi desenvolvido com foco em escalabilidade, utilizando uma arquitetura baseada em serviços para facilitar a manutenção e o isolamento de responsabilidades.

🧩 Estrutura de Serviços
Auth: Gestão de autenticação e controle de acesso seguro.

User: Gerenciamento completo de perfis e dados de usuários.

Password: Fluxos de segurança, recuperação e proteção de credenciais.

Checkout: Processamento de pedidos e lógica transacional da loja.

🔗 Integração
A API fornece suporte integral ao Front-end, garantindo persistência de dados e execução de regras de negócio em tempo real.
